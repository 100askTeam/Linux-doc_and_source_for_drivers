### 2. 建议

在《Linux系列教程之快速入门》的《嵌入式Linux应用开发基础知识》的视频里，
我们讲解了9节的输入系统应用编程：

```shell
1.输入系统框架及调试
2.现场编程读取获取输入设备信息
3.查询_休眠唤醒_方式读取输入数据
4.POLL_SELECT_方式读取输入数据
5.异步通知方式读取输入数据
6.电阻屏和电容屏
7.tslib框架分析
8.tslib交叉编译与测试
9.编写基于tslib的测试程序
```

强烈建议先学习这9个视频，至少必须学习其中这4个视频：

```shell
1.输入系统框架及调试
2.现场编程读取获取输入设备信息
3.查询_休眠唤醒_方式读取输入数据
6.电阻屏和电容屏
```
